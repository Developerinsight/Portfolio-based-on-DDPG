# Portfolio-based-on-DDPG
-----------------------------------------

## 재무제 분석을 통한 글로벌 투자 가이드
### 1. 퀀트와이즈 재무제표 기반으로 주식 종목 클러스터링(DTW clustering)
### 2. 클러스터링 특징 파악 및 투자자 성향과 매칭
### 3. 벤치마크(MSCI) 보다 높을 때까지 파라미터 조정
### 4. 강화학습(DDPG) 시뮬레이션 통해 Shape Ratio 가장 높은 최적의 포트폴리오 구성
